# Breast Cancer Prediction Using Machine Learning

## Project Overview
This project predicts whether a breast cancer tumor is benign or malignant using machine learning techniques.

## Algorithms Used
- Decision Tree Classifier
- Random Forest Classifier

## Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report
- ROC Curve (AUC)

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## How to Run

```bash
pip install pandas matplotlib seaborn scikit-learn joblib
py predictive_model.py
```

## Dataset
Breast Cancer Wisconsin Diagnostic Dataset (built into Scikit-learn).
